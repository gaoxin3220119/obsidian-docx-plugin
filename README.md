# obsidion-doxc-plugin

可以下载玩玩，源码就不放了，丢人

需要将 **docxjs.zip** 解压后保存到 .obsidian 目录下。注意不是放在 .obsidian\plugins 目录下面。

之前用过一个插件，作者没有写清楚，折腾半天：（

在 *Yaml* 区加入下面代码：

```
---
gx-docx: docx的绝对地址

gx-data: []
---
```

只支持批注，如果需要编辑，可以选择在菜单栏选择 “用默认Docx应用打开”来编辑
